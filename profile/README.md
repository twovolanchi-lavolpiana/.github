# 축구 전술 만들기 라볼피아나 입니다.
- 축구 선수를 등록하고, 움직임을 시뮬레이션하고 공유할 수 있는 기능입니다.

# 제작 배경
- 축구 및 축구 전술에 어릴 때부터 관심이 있었습니다.
- 사이드 프로젝트로 축구 전술을 시뮬레이션 해볼 수 있는 사이트를 개발하려는 목표를 세웠습니다.

# 기술 스택
- FrontEnd: TS / React
- BackEnd: Kotlin / Spring

# 진행 단계
- FE / BE 개발 진행 중입니다.
